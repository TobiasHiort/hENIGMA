# hENIGMA

hENIGMA is a simplified Enigma ciphering machine written in Haskell.

## Prerequisites
Install [Haskell](https://www.haskell.org/platform/).

## Running hENIGMA

Compile from `/src` with:
```
$ ghc -o hENIGMA hENIGMA.hs
```
Launch with:
```
$ ./hENIGMA
```
on Unix systems, or:
```
$ hENIGMA
```
on Windows systems.
